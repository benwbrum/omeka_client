# OmekaClient

A REST client for [Omeka][], using the API introduced in Omeka 2.1.

## Installation

Add this line to your application's Gemfile:

    gem 'omeka_client'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install omeka_client
